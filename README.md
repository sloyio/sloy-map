This library helps set up a beautiful geoportal using config. No code.

Features

1. Compatible with mapbox \ maplibre specifications
2. Multiple pages & layers
3. Vector tiles visualization
4. Filters
5. Themes
6. Localisation \ i18n

# Installation

npm i sloy-map

Check the examples/ folder
